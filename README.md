## Intuitive API

Swoldier is easy to use

```PHP
use Swoldier\Http\{Context, Router};

$server = new Swoole\Http\Server("0.0.0.0", 8080);
$router = new Router($server);

$router->get('/hello', function (Context $ctx) {
    $ctx->text("Hello, World!");
});

$server->start();
```
